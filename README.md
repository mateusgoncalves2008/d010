# Curiosidades de Tecnologia — Mascote do Android

Projeto simples em HTML/CSS que apresenta a história do mascote do Android. Este é o meu primeiro projeto utilizando Git, GitHub e GitHub Pages — utilizado para publicar uma versão estática deste site.

Visão rápida

-   Página única (HTML + CSS).
-   Objetivo: praticar HTML/CSS e fluxo básico de Git/GitHub.
-   Demo (GitHub Pages): https://mateusgoncalves2008.github.io/d010/ (habilite Pages se necessário)

Como executar localmente

1. Clone o repositório:
   git clone https://github.com/mateusgoncalves2008/d010.git
2. Entre na pasta do projeto:
   cd d010
3. Abra `index.html` no navegador ou sirva com um servidor local:
   python -m http.server 8000
   e acesse http://localhost:8000

Estrutura do projeto

-   index.html — conteúdo da página
-   style.css — estilos
-   Anexo/ — imagens e favicon
-   Font/ — fontes usadas

Sobre Git, GitHub e GitHub Pages (breve)

-   Este foi meu primeiro projeto com Git/GitHub: usei commits para registrar mudanças, branches para recursos e push para enviar ao GitHub.
-   Para publicar no GitHub Pages: vá em Settings → Pages → escolha a branch `main` e salve. A página ficará disponível em https://<seu-usuario>.github.io/d010/

Como contribuir

1. Fork este repositório.
2. Crie uma branch: `git checkout -b feature/nome-da-melhora`
3. Faça commits claros: `git commit -m "Descrição do que foi feito"`
4. Envie sua branch: `git push origin feature/nome-da-melhora`
5. Abra um Pull Request.
